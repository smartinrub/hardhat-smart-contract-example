# Hardhat Smart Contract Example

## Quickstart

```bash
cd hardhat-smart-contract-example
yarn
```

## Usage

### Deploy Contract

```
yarn hardhat run scripts/deploy.js
```

### Run Tests

```
yarn hardhat test
```

to run the tests with the gas reporter:

```
REPORT_GAS=true yarn hardhat test
```

### Access to Console

```
yarn hardhat console
```

### Clean Project

```
yarn hardhat clean
```
